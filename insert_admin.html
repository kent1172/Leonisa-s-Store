Add Admin: Mr. Vincent

Run this SQL in your Supabase SQL Editor to create the Admin account for Mr. Vincent. This statement handles both the authentication entry and the user metadata required by the app.

-- 1. Generate a new user in Supabase Auth
-- Email: vincent@leonisa.com
-- Password: VincentPassword2025 (Change this after login!)

INSERT INTO auth.users (
  id,
  email,
  encrypted_password,
  email_confirmed_at,
  raw_user_meta_data,
  aud,
  role
) VALUES (
  gen_random_uuid(),
  'vincent@leonisa.com',
  extensions.crypt('VincentPassword2025', extensions.gen_salt('bf')),
  now(),
  '{"role": "ADMIN", "username": "Mr. Vincent"}',
  'authenticated',
  'authenticated'
);

-- Note: The trigger created in schema.sql will automatically 
-- create the entry in public.profiles for this user.
Security Notice: After running this script, Mr. Vincent can log in using vincent@leonisa.com and the password VincentPassword2025. It is highly recommended to change the password immediately via the Supabase dashboard or an app "Reset Password" flow.

Step 2: Paste code and click 'Run'.
